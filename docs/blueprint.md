# **App Name**: WAEC Tutor AI

## Core Features:

- WAEC Question Explanation: Analyzes and explains WAEC past questions (Maths, English, Physics, Chemistry, Biology, etc.) by providing step-by-step explanations tailored for secondary school students.
- Quiz Generation: Generates WAEC-style quizzes with answer keys for various subjects. Uses reasoning tool to decide on suitable content.
- Audio Question Conversion: Converts student audio questions into text using Speech-to-Text API for analysis. Whisper transcription services.
- Text-to-Speech Feedback: Provides clear and concise voice responses using Text-to-Speech API (e.g., gpt-4o-mini-tts) for student feedback.
- Simplified Explanations: Detects when a student is confused and simplifies explanations automatically, ensuring clarity and understanding. Uses reasoning tool to identify student comprehension.
- User-Friendly Interface: Clean, intuitive design optimized for mobile devices with short paragraphs, bolded formulas, and minimal tables to ensure low-data usage and fast rendering.

## Style Guidelines:

- Primary color: A warm blue (#64B5F6) to inspire confidence and learning, appropriate for a light color scheme.
- Background color: Light, desaturated blue (#E3F2FD) for a calming and readable environment.
- Accent color: A vibrant green (#4CAF50), approximately 30 degrees away from the primary, to highlight correct answers, success messages, and call-to-action buttons.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for headlines and body text, for clear and accessible reading.
- Code font: 'Source Code Pro' for displaying code snippets if needed (e.g., in Physics or Mathematics examples).
- Use clear, simple icons to represent subjects (Maths, English, Science) and question types, aiding quick navigation.
- Mobile-friendly layout with a focus on readability; use of whitespace to separate content blocks.
